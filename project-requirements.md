Overview
In this project, you will design and build an interactive data visualization dashboard that
demonstrates your understanding of visual encoding principles, perceptual constraints, and
Gestalt layout design. You will choose a public dataset, create multiple coordinated views, and
conduct usability testing with peers to evaluate your design decisions.
Learning Objectives
• Apply the Expressiveness and Effectiveness principles to select appropriate marks and
channels
• Use Gestalt principles (proximity, similarity, enclosure) to organize a multi-view
dashboard layout
• Implement linked brushing and interactive filtering across coordinated views
• Evaluate design quality through structured usability testing with real users
• Reflect critically on design tradeoffs using visualization theory vocabulary
Requirements

1. Build an interactive dashboard using Tableau, D3.js, Plotly Dash, or Observable. The
   tool choice is yours, but the dashboard must be interactive (not static images).
2. Include at least 3 coordinated views with linked brushing or cross-filtering. Selecting
   data in one view must highlight or filter the corresponding data in all other views.
3. Encode at least 4 data dimensions using different visual channels (position, color,
   size, shape, etc.). Each channel must match the data type per the Expressiveness
   Principle.
4. Apply Gestalt principles to organize the layout. Related views should be grouped by
   proximity and/or enclosure. The layout should create a clear visual hierarchy.

Research Questions to answer using dashboard
What attributes most strongly associate with overall coffee quality?
Investigate which attributes (acidity, body, sweetness, bitterness) results in most strong correlation with the overall coffee quality score
Look into distribution by attribute
How do sensory profiles vary by region and species?
Analyze whether the region has an impact on the sensory profiles of the coffee bean
Does processing method affect coffee quality scores?
Analyze how processing methods affect quality attributes and see if there is a correlation between them
